# DP Dating

### How To Run This Project (for development)

- Get project by git clone
```
git clone..
```

- Download dependencies

```
cd dp-dating
npm install
```

- Add environment
```
cp .env.example .env
```

- Build and Run
```
docker-compose up --build -d
```

- Fresh logs
```
docker logs -f dating_app --tail=100
docker logs -f chatting_app --tail=100
```
